The workflow

Four steps, one command

A bundled Playwright scanner does the mechanical work in the page — reading computed styles and calling getAnimations() — so Claude reasons over real values instead of pixels.

1

Crawl

Discovers links, dedupes near-identical templates, and visits up to ~6 representative pages.

2

Capture

Screenshots each page at 3 breakpoints — desktop, tablet 768px, mobile 390px — plus full-page and scroll frames.

3

Extract

Pulls CSS custom properties, fonts, a usage-counted color palette, @keyframes, live WAAPI animations, transitions & hover rules.

4

Write

Claude reads the screenshots + JSON and writes design.md — every value cited from the real scan data.

Zero manual setup

Install once, scan anything

Requires Node 18+ and Claude Code. Playwright + Chromium install themselves the first time you run it.

1 · Add the skill

In the project where you want it — or add -g to install it user-wide.

npx skills add Shawnchee/design-scan

That's it. No Playwright install, no config. The first run downloads Chromium once (~300MB); every run after starts instantly.

2 · Give Claude a URL

A slash command, or just ask in plain language.

/design-scan https://linear.app # plain language works too: # "scan stripe.com and make a design.md" # "replicate this site's design: <url>"

1–4 minutes later you get design.md in your project root, plus the raw screenshots and JSON.

Overview

Stripe presents as a light, editorial, high-trust fintech brand. The canvas is predominantly white (#ffffff) with body copy in a cool desaturated slate (#425466, the single most-used color at 1,544 occurrences) and headings in a deep navy (#0a2540). The one loud gesture is a signature prismatic gradient "flame" — magenta → orange → periwinkle → violet — that sweeps through the hero. Personality is confident and restrained: light font weights (both body and headings render at 300), generous whitespace, and motion that is quick and functional rather than showy.
